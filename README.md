# SmartplannerV1
완벽한 스터디 플래너를 만드세요~
<br />
시험 기간은 다가오는데 어떤 과목부터 몇 시간 동안 공부해야 할지 막막한 학생들을 위해 시험 날짜, 목표 등급, 현재 진척도(아는 정도)만 입력하면 하루 공부 루틴을 10분 단위로 짜주는 스터디 플래너입니다.
<br />
⭐️Key Features
<br />
시험기간&주말 자동 스케줄링: 시험 첫날, 마지막날을 선택하면 자동으로 스케줄을 빌드합니다.
<br />
목표 등급&중요도 맞춤 가중치 알고리즘: 높은 목표 등급을 가졌거나 진도가 느린 과목에 더 많은 시간을 배분하게 가중치를 두었습니다.
<br />
일별 스터디 플래너&카운트다운: 달력에서 날짜를 클릭하면 그날 하루 동안 과목별로 공부해야 하는 시간을 직관적으로 보여줍니다. 시험 이틀 전부터는 카운트다운 배지가 등장합니다. 
<br />
자동 저장&백업코드: 로컬 스토리지가 활성화되어 있습니다. [일별 스케줄 텍스트 복사] 버튼으로 내용을 복사해 메모장에 기록할 수 있습니다. [단축 백업코드 복사/불러오기] 버튼으로 일정을 다른 기기로 가져갈 수 있습니다.
<br />
⭐️Tech Stack
<br />
Library: React.js(v18)
<br />
Styling: Tailwind CSS
<br />
Icons: Lucide Icons
<br />
Runtime: HTML5 LocalStorage, Babel standalone
